# CareerHub-backend

RESTful API backend using Laravel

## Installation Instructions

-   Install Dependencies
    ```bash
    composer install
    ```

*   Create .env file

    ```bash
    cp .env.example .env
    ```

*   Generate encryption key

    ```bash
    php artisan key:generate`
    ```

*   Migrate DB

    ```bash
    php artisan migrate
    ```

*   Seed DB

    ```bash
    php artisan db:seed
    ```

-   Enjoy ^\_^
